# shipper-demo
货运后台框架  
使用技术：
vue-cli+Vue2.x + Vuex + Vue-router
组件
less
Superagent
echart
element框架（风格，日历，分页）

解压 在webstrom中打开
cd logistic-project
npm install
npm run dev

  npm install webpack-zepto

	npm run dev
